== Description ==



Put a clock on your blog or website in any timezone of your choosing, use it as a widget, as a Visual Composer Block, or include it in your content using the generated shortcode.  
Blog Clock is incredibly easy to use and fully customizable. You can create as many indipendent clock as you wish, each with different timezone and different styling options, such as coloring, fonts, and layout. 
Blog clock comes with two different layouts set, analog and digital, each of them will provide you with spefic styling options and features. 
All the clocks updates in real time and are perfectly syncronized.



FUNCTIONS



- Use the fully customizable new Blog Clock Post Type to create as many clocks as you wish and use them indipendently in your site.
- Display the clock anywhere on your website using the generated shortcode, the Visual Composer Block or the specific widget.
- Ability to choose timezone to display your desired time or to automatically adjust the time depending on the client timezone.
- Ability to customize the style of your clock indipendently.
- Two main clock layout, analog and digital.
- Change the colors, the fonts and many other styling settings.
- Choose between many different time format for your clock Timestamp is also available! 
- Choose between many different date format, if you need a specific format, add your custom one follwing this specifications!
- Choose a specific width or to adjust to its container width.
- Set alignment. Left, center, right.




== Installation ==



1. Select 'Add new' in the 'Plugins' menu to find and install the latest Blog clock plugin version.

2. Activate the plugin.

3. Settings are under “Blog Clock” in the tools menu.



== Frequently Asked Questions ==

= Can I use multiple clocks on the same page =

Yes.

= Can I set multiple timezone for different clocks on the same page? =

Yes, you can set as many clocks with specific timezones as you want at the same time

= Does this plugin updates the seconds in realtime? =

Yes, both the digital and the analog clocks work in real-time for seconds and milliseconds

= Does this plugin support Visual Composer WP Backery =

Yes, it will create a custom VC Block to add converters to your content

= Can I use this plugin as a Widget? =

Yes, there’s a specific Widget ready to be used in your Widget section
